# Loker Skills

This repository stores the custom Codex skills installed under:

`/Users/tujiajun/.codex/skills`

System-managed skills in `.system/` are intentionally ignored. When adding or updating custom skills, commit and push changes from this directory:

```sh
cd /Users/tujiajun/.codex/skills
git status
git add .
git commit -m "Sync custom skills"
git push
```
